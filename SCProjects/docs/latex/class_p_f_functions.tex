\subsection{Detailed Description}
Helper class for the \hyperlink{class_navigation_agent}{Navigation\-Agent}. Contains methods to calculate the potentials generated by own units, enemy units and neutral objects in specific points.

Author\-: Johan Hagelback (\href{mailto:johan.hagelback@gmail.com}{\tt johan.\-hagelback@gmail.\-com})
