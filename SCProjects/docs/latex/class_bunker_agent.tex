\subsection{Detailed Description}
The \hyperlink{class_bunker_agent}{BunkerAgent} handles Terran Bunker buildings.

Implemented abilities:
\begin{DoxyItemize}
\item Keeps track of the units inside the bunker.
\item Which units to use in the Bunker depends on the enemy race(s).
\end{DoxyItemize}
