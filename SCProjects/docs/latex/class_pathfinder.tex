\subsection{Detailed Description}
This class is used to find a path betweed two tiles in the game world. Currently it uses the A-\/star implementation in B\-W\-T\-A, but it can easily be changed to another algorithm if needed.

The pathfinder is threaded, so agents have to request a path that is put in a queue. Agents have to check the is\-Ready() method to find out when the path finding is finished.

Author\-: Johan Hagelback (\href{mailto:johan.hagelback@gmail.com}{\tt johan.\-hagelback@gmail.\-com})
