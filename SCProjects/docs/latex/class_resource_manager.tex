\subsection{Detailed Description}
\hyperlink{class_resource_manager}{Resource\-Manager} handles the resources and where to spend them. An agent must ask the \hyperlink{class_resource_manager}{Resource\-Manager} for permission to build/upgrade/research before issuing the order.

The \hyperlink{class_resource_manager}{Resource\-Manager} is implemented as a singleton class. Each class that needs to access \hyperlink{class_resource_manager}{Resource\-Manager} can request an instance, and all classes shares the same \hyperlink{class_resource_manager}{Resource\-Manager} instance.
