\subsection{Detailed Description}
The \hyperlink{class_medic_agent}{Medic\-Agent} handles Terran Medics.

Implemented special abilities\-:
\begin{DoxyItemize}
\item Targets and follows own organic units that are in the attacking force.
\item Heals nearby damaged organic units.
\end{DoxyItemize}
