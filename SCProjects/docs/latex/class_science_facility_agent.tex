\subsection{Detailed Description}
The \hyperlink{class_science_facility_agent}{ScienceFacilityAgent} handles Terran Science Facility buildings.

Implemented abilities:
\begin{DoxyItemize}
\item Builds the Covert Ops addon to support Ghosts.
\item Can build the Physics Lab addon to support Battlecruisers.
\item Research the EMP Shockwave and Irradiate abilities for Science Vessels.
\end{DoxyItemize}
