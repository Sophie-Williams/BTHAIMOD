\subsection{Detailed Description}
The \hyperlink{class_exploration_manager}{Exploration\-Manager} handles all tasks involving exploration of the game world. It issue orders to a number of units that is used as explorers, keep track of areas recently explored, and keep track of spotted resources or enemy buildings.

The \hyperlink{class_exploration_manager}{Exploration\-Manager} is implemented as a singleton class. Each class that needs to access \hyperlink{class_exploration_manager}{Exploration\-Manager} can request an instance, and all classes shares the same \hyperlink{class_exploration_manager}{Exploration\-Manager} instance.
