\subsection{Detailed Description}
The \hyperlink{class_constructor}{Constructor} class is resonsible for constructing the buildings added to the build queue.

The \hyperlink{class_constructor}{Constructor} is implemented as a singleton class. Each class that needs to access \hyperlink{class_constructor}{Constructor} can request an instance, and all classes shares the same \hyperlink{class_constructor}{Constructor} instance.

When a request to construct a new building is issued, no construction are allowed until the worker has moved to the buildspot and started constructing the building. This is to avoid that the needed resources are not used up by other build orders. During this time the \hyperlink{class_constructor}{Constructor} is locked, and new construction can only be done when unlock has been called.
