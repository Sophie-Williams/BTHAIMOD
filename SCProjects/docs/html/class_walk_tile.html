Detailed Description

This class represents a Starcraft walk tile. Walk tiles are 8 positions/0.25 terrain tile.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
