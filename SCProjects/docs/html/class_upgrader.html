Detailed Description

Upgrader handles upgrades and techs. Add the requested upgrade and/or tech to the upgrader and the bot executes it when enough resources are available.

The Upgrader is implemented as a singleton class. Each class that needs to access Upgrader can request an instance, and all classes shares the same Upgrader instance.
