Detailed Description

This class handles the main AI loop that is executed each frame. It is separated from the BTHAIModule::onFrame() to make it possible to use multi-threading.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
