Detailed Description

This class is used to find a path betweed two tiles in the game world. Currently it uses the A-star implementation in BWTA, but it can easily be changed to another algorithm if needed.

The pathfinder is threaded, so agents have to request a path that is put in a queue. Agents have to check the isReady() method to find out when the path finding is finished.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
