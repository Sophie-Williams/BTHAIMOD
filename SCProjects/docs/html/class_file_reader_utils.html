Detailed Description

This class contains some common methods used by classes handling the reading of buildorder/techs/upgrades/squad setup files.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
