Detailed Description

The StructurePositioner class is used to find positions to place new buildings at. It keeps track of which map tiles are occupied (by own buildings) or unavailable (blocked by terrain, minerals patches etc.).

Internally a matrix of the same size as the map is used. If a Tile is occupied or cant be reached by ground units, the value if the tile is 0. If the Tile can be built on, the value is 1. Buildings typically use up more space in the matrix than their actual size since we want some free space around each building. Different types of buildings have different space requirements.

The StructurePositioner is implemented as a singleton class. Each class that needs to access StructurePositioner can request an instance, and all classes shares the same StructurePositioner instance.
