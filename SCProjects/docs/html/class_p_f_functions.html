Detailed Description

Helper class for the NavigationAgent. Contains methods to calculate the potentials generated by own units, enemy units and neutral objects in specific points.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
