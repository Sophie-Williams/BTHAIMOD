Detailed Description

When a game is started a strategy is selected depending on the map and, if known, the opponent race. After each game the result is stored to a statistics file (bwapi-data/AI/BTHAI/StrategyStats.csv). Strategies that previously have been successful have a higher probability of being selected, but all strategies have at least 15% chance of being used.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
