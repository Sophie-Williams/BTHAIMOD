Detailed Description

Helper class for Profiler. This class represents a profiling of one specific codeblock. Profiling can be done on any number of codeblocks.

Author: Johan Hagelback (johan.nosp@m..hag.nosp@m.elbac.nosp@m.k@gm.nosp@m.ail.c.nosp@m.om)
